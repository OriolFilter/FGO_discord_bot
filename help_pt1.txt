#FGO_bot

**Commands**

```ini
[Events]        [events|aevents|fevents]

[Classes]       [class|archer|saber|lancer|caster|rider|assassin|berserker|
                 shielder|ruler|avenger|mooncancer|foreigner|beast]

[Misc]          [help|time|invite]
```
**Events**

```ini
    [f.events]    [Returns a link to a event list (USA)]

    [f.aevents]   [Returns a list of 'active' events and a short note
                   with a link to check information from each event]

    [f.aevents]   [Returns a list of 'future' events and a short note
                   with a link to check information from each event]
```
**Class**

```ini
    [f.class]         [Returns a list of classes aviable and their respective links]
    [f.saber]         [Returns his respective class link]
    [f.archer]        [Returns his respective class link]
    [f.lancer]        [Returns his respective class link]
    [f.rider]         [Returns his respective class link]
    [f.caster]        [Returns his respective class link]
    [f.assassin]      [Returns his respective class link]
    [f.berserker]     [Returns his respective class link]
    [f.shielder]      [Returns his respective class link]
    [f.ruler]         [Returns his respective class link]
    [f.avenger]       [Returns his respective class link]
    [f.mooncancer]    [Returns his respective class link]
    [f.foreigner]     [Returns his respective class link]
    [f.beast]         [Returns his respective class link]
```
**Misc**

```ini
    [f.time]      [Returns current time of the server (PST hour)]
    [f.invite]    [Returns a link to invite this bot to your own discord-server]
    [f.help]      [Returns help message]
```
